# jc.parsers.pip\_list

jc - JSON Convert `pip-list` command output parser

Usage (cli):

    $ pip list | jc --pip-list

or

    $ jc pip list

Usage (module):

    import jc
    result = jc.parse('pip_list', pip_list_command_output)

Schema:

    [
      {
        "package":     string,
        "version":     string,
        "location":    string
      }
    ]

Examples:

    $ pip list | jc --pip-list -p
    [
      {
        "package": "ansible",
        "version": "2.8.5"
      },
      {
        "package": "antlr4-python3-runtime",
        "version": "4.7.2"
      },
      {
        "package": "asn1crypto",
        "version": "0.24.0"
      },
      ...
    ]
